# design — DayDot UI Spec (1:1 复刻基线)

> 本文档以“截图为唯一真相”。任何实现（无论技术栈）必须做到视觉与交互 1:1 复刻。
> 设计统一通过 token 管理，禁止在组件里散落硬编码尺寸/颜色。

## 设计目标

- 移动优先（竖屏安卓 Chrome）
- 莫兰迪低饱和、米白基调
- 卡片化模块、轻阴影、超圆角胶囊控件
- 交互克制：点击空白关闭弹层/下拉；主路径极少跳转阻塞

## 页面清单（以截图命名）

- 记录（卡片墙）
- 录入（表单卡片）
- 日历（月历 + 空状态）
- 标签详情（时间线）
- 日历（带 Tag 筛选态：顶部“已过去 X 天”卡 + 月历 + 当日记录）
- 全部记录（列表 + 顶部筛选）
- 标签管理（父级/子级输入 + 标签列表）

## 组件清单（必须复用同一套视觉 token）

- `AppShell`：页面背景、统一左右内边距、顶部安全区、底部导航占位
- `BottomNav`：三段式（记录 / + / 日历）
- `ComboSearch`：统一搜索/选择框（可输入 + 可选择的 combobox，下拉锚定在输入框正下方）
- `Card`：白底、统一圆角、统一轻阴影、统一内边距
- `Pill`：胶囊（筛选、Tag 展示）
- `IconButton`：圆形按钮（齿轮、编辑、删除、加号、左右箭头）
- `PrimaryButton`：底部大按钮（“记录”）
- `DaysSinceCard`：大号数字居中（“已过去 XX 天”）
- `CalendarPopover`：统一日期选择（自定义月历弹层，不使用系统原生日期选择器）
- `Timeline`：竖线 + 节点 + 记录卡
- `RecordCard` / `RecordRow`：卡片墙条目 / 列表条目（右侧操作按钮一致）
- `TagManagerPanel`：父级/子级输入行 + 说明文字 + 标签列表卡

## 设计 token（必须集中定义并引用）

> 说明：具体数值将以截图为准在实现阶段校准；实现时允许“token 内部”使用固定 px，但组件内只引用 token。

### 色彩

- `--bg`: #FAF6F1
- `--card`: #FDF9F4
- `--secondary`: #F5EDE4
- `--accent`: #C4A882
- `--text`: #4A4543
- `--text-sub`: #9B918A
- `--border`: #E8E0D8

Tag 色（父级 10 色）：延续 v1 莫兰迪色谱；子级继承父级并派生浅底。

### 圆角

- `--r-pill`: 胶囊/搜索条/输入行
- `--r-card`: 卡片/容器
- `--r-fab`: 圆形按钮（含底部“+”）

### 阴影

- `--shadow-card`: 卡片轻阴影（暖灰、低不透明度、柔和 blur）
- `--shadow-float`: 弹层/下拉（略强但仍克制）

### 间距与尺寸

- `--space-page-x`: 页面左右内边距
- `--space-section-y`: 模块上下间距
- `--space-card-gap`: 卡片间距
- `--control-h`: 输入/选择行统一高度
- `--nav-h`: 底部导航高度
- `--fab-size`: 底部“+”直径
- `--icon-btn-size`: 圆形 icon button 直径

### 组件级 token（统一所有搜索框 / 日历）

- `--combo-h`: combobox 输入行高度（与其它输入行一致）
- `--combo-pad-x`: combobox 左右内边距
- `--combo-icon-size`: combobox 内图标尺寸
- `--combo-menu-gap`: 输入框与下拉层间距（极小）
- `--combo-menu-max-h`: 下拉层最大高度（超出可滚动）
- `--calendar-cell-size`: 日历单元格尺寸
- `--calendar-dot-size`: 记录圆点尺寸
- `--calendar-selected-bg`: 选中日期圆底色（通常为 accent）
- `--calendar-footer-h`: 日历底部操作条高度（“清除/今天”）
- `--modal-overlay`: 弹窗遮罩色/透明度（编辑记录等）

### 排版

- `--font-title`: 卡片大标题
- `--font-days`: “2/4 天”大数字
- `--font-body`: 常规正文
- `--font-caption`: 辅助信息（日期、说明）
- `--weight-strong`: 标题粗体

## 交互一致性（验收要点）

- 下拉/弹层：点击空白关闭；下拉层必须锚定在触发控件正下方展开
- **所有“搜索框”统一为 combobox**：可输入过滤 + 可点击选择；支持展示已选 Tag（含父级 pill）；右侧可有圆形按钮（如齿轮）
- **所有“日历”统一为自定义月历弹层**：居中浮层、左右切月、网格与圆点标记一致；底部操作含“清除 / 今天”
- 底部导航：当前 Tab 高亮更深；中央“+”在不同页存在明暗切换（截图为准）
- 日历：有记录日期显示圆点；选中日期为实心圆底
- 列表操作：编辑/删除均为右侧圆形 icon button（尺寸/间距一致）
