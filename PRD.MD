# PRD — DayDot v2

## 项目简介
DayDot 是一个纯前端个人 Web App，记录生活中不规律事件的日期，快速查看"上次什么时候、已过去多少天"。数据存浏览器本地，零网络请求，离线可用。仅个人使用，仅记录文字（无图片）。

## 功能需求

### 录入页（底部 Tab "＋"）
- [x] 选择 Tag（搜索已有二级 Tag；常用标签 Top 9 快捷选择，3×3 网格）
- [ ] Tag 行右侧「＋」跳转标签管理页新建（录入页不直接创建 Tag）
- [x] 填写备注（可选，纯文字）
- [x] 选择日期（默认今天，可改过去日期）[实现调整: 采用自研日历弹层，非系统原生日期选择器]
- [x] 确认保存后清空 Tag 与备注
- [x] 常用标签 Top 9：按**使用频率**排序（出现次数高优先；并列再按最近使用时间）

### 记录页/仓库（底部 Tab "记录"）
- [x] 首屏为 Tag 卡片墙（仅有记录的叶子/独立 Tag，显示距今天数 + 父级颜色）
- [x] 点击卡片进入该 Tag 时间线详情（竖线+节点布局），顶部"已过去 XX 天"卡片
- [x] 右上角"全部"入口查看全量记录（按时间倒序）
- [x] 顶部搜索框（关键词搜备注/名称）+ 日期筛选（支持 Tag 筛选）
- [x] 记录编辑（弹层，改日期/Tag/备注）和删除（二次确认）
- [x] 选一级 Tag 筛选时自动包含其二级 Tag
- [x] 排序：卡片墙按**最近发生时间**倒序（越新越靠前）
- [x] 排序：全部记录按 `eventDate` 倒序；同日按 `updatedAt` 倒序

### 日历页（底部 Tab "日历"）
- [x] 月历视图，有记录的日期显示圆点
- [x] 点击某天展开该日所有记录
- [ ] 可选 Tag 下拉筛选：选中后圆点用该标签颜色，并显示"已过去 XX 天"；不选等同全局
- [x] 排序：某一天内的记录按 `updatedAt` 倒序（越新越靠前）

### 标签管理页（从录入页/记录页跳入）
- [x] Tag 新建（父级可下拉选择，子级输入填写）、改名
- [x] 两级层级（父级→子级），每条记录必须有父级和子级（自动一致性清理：无子级父级/无父级子级会被删除）
- [x] 父级可设颜色（10 个莫兰迪色），子级继承父级颜色
- [x] 删除可选：仅删 Tag（记录保留但 Tag 置空）或连带删记录（弹窗三选一）
- [x] 显示 Tag 出现次数（子级频次）
- [x] 列表卡片化 + 搜索过滤
- [x] 排序：同父级的子级标签聚合展示；整体与组内均按**使用频率**倒序（并列再按最近使用时间）

### 数据备份（设置/标签管理页入口）
- [x] 导出 JSON 备份文件
- [x] 导入 JSON 恢复数据（导入前自动导出一份当前备份，避免误覆盖）

### PWA
- [x] Service Worker 离线缓存，断网可用
- [ ] 可添加到手机桌面，standalone 模式
- [x] 新版本发布后提示刷新（提示后可一键刷新生效）

## 体验要求
- 移动优先（竖屏安卓 Chrome 主场景）
- 莫兰迪低饱和色系，米白基调
  - 背景 #FAF6F1，卡片 #FDF9F4，次级/胶囊 #F5EDE4（避免纯白）
  - 强调 #C4A882，主文字 #4A4543，辅助 #9B918A，分割线 #E8E0D8
- "已过去 XX 天"大数字居中，视觉焦点
- Tag 展示统一为：子级名 + 父级胶囊（同字号同高度）
- 弹窗/下拉框点击空白关闭，确认弹窗风格统一
- 搜索/选择入口统一为“可输入 + 可选择（可搜索过滤）”的组合下拉，并且下拉层必须在原控件正下方展开
- 输入/选择行统一行高 + 字号 + 浅米白底色
- 无记录/无标签时显示空状态引导

## 怎样算做完
手机 Chrome 完成录入/记录/日历/标签管理全流程，离线可用，可添加到桌面，支持导入导出备份。
